
#  Basic to Advanced NLP (Natural Language Processing)

This repository showcases my practical exploration of Natural Language Processing and Generative AI using Python and transformer-based models. 
The projects demonstrate progressive learning from basic text generation to building structured, context-aware AI systems.

Through these implementations, I gained hands-on experience with pretrained language models, prompt engineering, controlled text generation, 
and combining AI with logical dialogue management to create coherent and meaningful outputs.
